Exercício 10 – Constante obrigatória
Crie um programa que:
1. Declare uma constante chamada LIMITE_MAXIMO com valor
100.
2. Declare uma variável chamada valorAtual.
3. Atribua 80 a valorAtual.
4. Depois atribua 100 a valorAtual
5. Exiba os dois valores.
Após declarar a constante LIMITE_MAXIMO com o valor 100, é possível
atribuir um novo valor a ela? Tente fazer essa alteração em código.

programa {
  funcao inicio() {
    const inteiro LIMITE_MAXIMO=100
    inteiro ValorAtual=80
    escreva("Valor Atual: ",ValorAtual,"\n")
    ValorAtual=100
    escreva("Valor Atual: ",ValorAtual,"\n")
    //NÃO É POSSIVEL ALTERAR O VALOR DE UMA CONSTANTE
  }
}

Identifique o erro:
inteiro idade = 18.5 // Não pode inicializar uma variável tipo inteiro com um valor real
real altura = "1.70" // Uma variável real não pode receber um valor tipo cadeia
caracter letra = "A" // Não se usa aspas duplas para variável tipo caracter
